# UNG for VS Code

Freelance billing and time tracking inside VS Code. No spreadsheets. No browser tabs.

## Install

1. Install the CLI:
   ```bash
   brew install andriiklymiuk/tools/ung
   ```

## Getting Started

Click the **UNG icon** in the Activity Bar → Initialize → Add company → Add client → Start tracking.

## Key Shortcuts

| Action | Shortcut |
|--------|----------|
| Start tracking | `Cmd/Ctrl + Alt + T` |
| Stop tracking | `Cmd/Ctrl + Alt + S` |
| Create invoice | `Cmd/Ctrl + Alt + I` |
| Invoice from time | `Cmd/Ctrl + Alt + G` |
| Command center | `Cmd/Ctrl + Shift + U` |

## Daily Workflow

1. **Start** → `Cmd+Alt+T` → pick contract → work
2. **Stop** → `Cmd+Alt+S` → time saved
3. **Invoice** → `Cmd+Alt+G` → generate from tracked hours → send

## Features

- **Dashboard** — revenue, projections, quick stats
- **Time tracking** — timer in status bar, manual entry
- **Invoices** — create, PDF export, email to clients
- **Contracts** — hourly, fixed price, retainer
- **Expenses** — track business costs

## Troubleshooting

**CLI not found?** Run `ung --version` in terminal. If missing, install via brew and restart VS Code.

**Views not loading?** Try `UNG: Refresh All Views` from Command Palette.
