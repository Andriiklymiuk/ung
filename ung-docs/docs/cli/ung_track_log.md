## ung track log

Log time manually (without timer)

### Synopsis

Manually log hours worked on a contract or client

```
ung track log [flags]
```

### Options

```
      --client string    Client name (e.g., humbrella)
      --contract int     Contract ID
  -h, --help             help for log
      --hours float      Hours worked
      --notes string     Session notes
      --project string   Project name
```
