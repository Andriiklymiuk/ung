## ung company edit

Edit an existing company

```
ung company edit [id] [flags]
```

### Options

```
      --address string        Company address
      --bank-account string   Bank account number (e.g., IBAN)
      --bank-name string      Bank name
      --bank-swift string     Bank SWIFT/BIC code
      --email string          Company email
  -h, --help                  help for edit
      --name string           Company name
      --tax-id string         Tax ID
```
