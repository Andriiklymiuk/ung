## ung contract add

Add a new contract

```
ung contract add [flags]
```

### Options

```
      --client int        Client ID
      --currency string   Currency (default "USD")
  -h, --help              help for add
      --name string       Contract name
      --price float       Fixed price (for fixed_price contracts)
      --rate float        Hourly rate (for hourly contracts)
      --type string       Contract type (hourly, fixed_price, retainer)
```
