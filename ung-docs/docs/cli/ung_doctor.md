## ung doctor

Check the health of your ung installation

### Synopsis

Run diagnostics on your ung installation to identify and fix common issues.

This command will check:
- Database connection and schema
- Configuration files
- Required directories and permissions
- Data integrity

```
ung doctor [flags]
```

### Options

```
  -h, --help   help for doctor
```
