## ung invoice new

Create a new invoice

```
ung invoice new [flags]
```

### Options

```
      --client int           Client ID
      --client-name string   Client name (partial match, min 3 chars)
      --company int          Company ID
      --currency string      Currency code (default "USD")
      --description string   Invoice description
      --due string           Due date (YYYY-MM-DD)
  -h, --help                 help for new
      --price float          Invoice amount (required)
```
